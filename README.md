## Running Locally

If you clone this repo instead of using the release URL you'll need to build the platform once:
```sh
./jump-start.sh
roc build.roc --linker=legacy
```
Then you can run like usual:
```sh
$ roc examples/hello-world.roc
Hello, World!
```
